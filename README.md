# React-Native
React - Redux - Flow - Eslint - Logger

# Install

1. **Clone Project**
```
  $ git clone https://github.com/vsarikaya/react-native.git
  $ cd react-native
```

2. **Install Dependencies**
```
  $ npm install
```

3. **Running Android**
```
  $ npm run android
```

4. **Running IOS**
```
  $ npm run ios
```

# Linting Code
```
  $ npm run lint
```

# Flow
```
  $ npm run flow-start 
  $ npm run flow-stop
  $ npm run flow-status
  $ npm run flow-coverage
```
